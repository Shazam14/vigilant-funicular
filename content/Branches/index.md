# Branches

Kedco FX operates across **9 branches** in Cebu, Philippines. Each branch has a unique code and address, and operates with its own shifts, floats, and daily positions.

See [[Branches/Branch List]] for the full address table.

## How Branch Tracking Works

- Device is assigned a branch once via the [[Features/Terminal & Branch Select|branch + terminal selector]]
- `branch_id` is stamped on every [[Features/Currency Exchange|transaction]] and [[Features/Shift Management|shift]]
- [[Features/EOD Reports|EOD reports]] and [[Features/Dashboard|dashboard]] data are filterable per branch
