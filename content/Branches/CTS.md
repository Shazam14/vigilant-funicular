# CTS Branch (CTS)

**Branch code:** `CTS`
**Address:** A-218 City Timesquare, Mantawe Ave., Mandaue City, Cebu

## Overview

CTS is Kedco FX's **City Timesquare** branch, located inside a commercial mall area in Mandaue City. It primarily serves walk-in retail customers at the mall.

## Operations

- Standard [[Roles/Cashier|cashier]] + [[Roles/Rider|rider]] setup
- [[Features/Shift Management|Shifts]] opened and closed daily with the branch stamped on every transaction
- [[Features/EOD Reports|EOD reports]] rolled up per branch for admin review

## Terminals

Devices are configured via the [[Features/Terminal & Branch Select|branch + terminal selector]]:
- Counter 1, Counter 2, Counter 3
- Rider Phone
- Supervisor Tablet
