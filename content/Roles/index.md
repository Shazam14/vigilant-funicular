# Roles

Kedco FX has four user roles, each with different access and responsibilities.

- [[Roles/Admin|Admin]] — Full access, branch oversight, EOD reports
- [[Roles/Cashier|Cashier]] — Processes transactions, manages shifts
- [[Roles/Rider|Rider]] — Carries currency between branches/locations
- [[Roles/Dev|Dev]] — System access for testing and development
