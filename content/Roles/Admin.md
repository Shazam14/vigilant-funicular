# Admin

The admin role has full visibility and control over branch operations.

## Responsibilities

- Review and approve [[Features/EOD Reports|EOD reports]]
- Monitor the [[Features/Dashboard|dashboard]] — PHP cash, capital trend, active shifts
- View all [[Features/Transaction Log|transactions]] across all cashiers
- Authorize [[Features/Replenishment|replenishments]]
- Manage users and branch settings

## Access

- All [[Branches/Branch List|branches]] (depending on assignment)
- Full [[Features/Transaction Log|transaction log]]
- [[Data/Daily Positions|Daily positions]] and [[Data/Stock Summary|stock summaries]]
- User management
